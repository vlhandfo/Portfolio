Titanic Data Set: Exploratory Data Analysis

Starting point: Make School Tutorial

Data set: Titanic - Machine Learning from Disaster

Notes:

  • I used the tutorial from Make School as a starting point and expanded from there.
  • I renamed the “train.csv” file to “titanic.csv” for simplicity and is found in the “data” directory. (Relative file path: “data/titanic.csv”)
  • This analysis requires the following packages:
    • numpy
    • pandas
    • matplotlib
    • seaborn

Features (that need explanation)

Feature Definition Key
Survived survival 0 = no; 1 = yes
Pclass Ticket Class 1 = 1st; 2 = 2nd; 3 = 3rd
SibSp # of sbilings/spouses aboard int
Parch¹ # of parents / children aboard int
Embarked Port of Embarkment C = Cherbourg; Q = Queenstown; S = Southampton

Notes:
¹ Some children travelled only with a nanny, therefore \(parch = 0\) for them.
² Age is a fraction if less than 1

We found a correlation between Sex and Survival, but not between Age and Survival.

  • Only about 59 percent of children survived which is lower than the overall survival rate (61 percent), and the women-and-children survival rate (71 percent)
# Probability of women's survival
women = df[(df['Sex'] == 0) & (df['Age'] >= 16)]
surviving_women = df[(df['Sex'] == 0) & (df['Age'] >= 16) & (df['Survived'] == 1)]

women_survival_probability = np.round(surviving_women.shape[0] / women.shape[0], 4)
women_survival_probability

# Probability of men's survival

Observation

Women actually had the best chance of surviving and children had a pretty good chance, but men did not.

  • Less than 1 in 5 men on the Titanic, (\(< 20 \%\)), survived
  • Almost 4 out of 5 women, (\(\approx 80 \%\)), survived.
